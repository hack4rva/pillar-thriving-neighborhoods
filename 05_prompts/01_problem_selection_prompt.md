# Problem Selection Prompt

Read the targeted problem statements and core files in this repo and recommend one problem to pursue that is both Richmond-specific and weekend-buildable.

Produce:
1) A ranked list of 2–3 candidate problems with rationale.
2) For the top pick: user, workflow, public data/doc base, demoable artifact by Sunday.
3) A short "Why not the others" section.
4) A risk list with mitigation ideas.

Constraints
- No eligibility/legal determinations.
- No policy/integration dependencies.
- Must cite which repo passages or public sources support the choice.
- Housing Compliance (Statement 1) has a data-not-ready flag — acknowledge this in your analysis.
- Neighborhood Development Discovery (Statement 2) scores 26/32 and has a clearer data path via Legistar and GeoHub.

Output format
- Decision memo suitable to paste into `99_templates/decision_memo.md`.
