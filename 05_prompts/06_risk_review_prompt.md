# Risk Review Prompt

Review our proposed project concept for the Thriving Neighborhoods pillar.

Identify:
- factual assumptions not verified by this repo
- data dependencies and quality concerns (especially: is Legistar API access confirmed for Richmond?)
- legal/ethical risks (eligibility, housing advice, privacy, implied City endorsement)
- ways the project could mislead users or overclaim impact (e.g., implying a development will or won't be approved; implying AI summaries are authoritative)
- scope cuts that make the prototype safer and more credible
- any claims about displacement, housing affordability, or planning outcomes that require evidence beyond what is in this repo

Rewrite the concept into a safer, sharper, more buildable version that keeps source links and disclaimers front-and-center.

Special considerations for this pillar:
- Plain-language summaries of planning documents must be labeled as interpretations, not official translations
- Development maps must show data freshness and link to Legistar source for each item
- Compliance tools must not make compliance determinations — only surface information for staff review
- Homeownership affordability tools must not claim to calculate eligibility for programs
